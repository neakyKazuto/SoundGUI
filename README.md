# Приложение на паттерне MVVM
## Краткая сводка
**Наименование:** SoundGUI

**Краткое описание:**
Программа воспроизводит звук по количеству строк в текстовом поле.
****
## Структура программы
- Главная
  - Кнопка запуска (или F11)
  - Кнопка отмены
  - Статус бар (выводится текущая итерация)
  - Текстовое поле

- Настройки
  - Ползунок громкости
****
